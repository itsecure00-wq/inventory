# @87 语言切换 — 设计文档

**日期:** 2026-02-24
**范围:** `CheckPage.html` 只改这一个文件
**功能:** 中文 ↔ 缅甸文切换，偏好存入 `localStorage`

---

## 背景

系统员工中有缅甸籍工人，他们的主要操作页面是盘点页（CheckPage）。
目前全部 UI 文字为中文，需要支持切换到缅甸语，以降低操作门槛和出错率。

---

## 决策记录

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 哪些页面 | 只有 CheckPage | 员工只用盘点页；AdminPage/Dashboard 由主管使用 |
| 偏好保存 | `localStorage` | 简单，per-device，无需后端改动 |
| 切换按钮位置 | 顶栏右侧，退出按钮左边 | 易发现，不占主内容区空间 |
| 实现方案 | 方案 A：`data-zh/my` 属性 + `t(key)` 字典 | 静态和动态内容分开处理，改动范围清晰 |

---

## 架构

### JS 核心

```javascript
// 在 <script> 顶部加入
var currentLang = localStorage.getItem('ims_lang') || 'zh';

const STRINGS = {
  zh: { /* ~80 条中文字符串 */ },
  my: { /* ~80 条缅甸文字符串 */ }
};

function t(key) {
  return (STRINGS[currentLang] && STRINGS[currentLang][key])
    || (STRINGS.zh[key])
    || key;
}

function setLang(lang) {
  currentLang = lang;
  localStorage.setItem('ims_lang', lang);
  applyLang();
}

function applyLang() {
  document.querySelectorAll('[data-zh]').forEach(function(el) {
    el.textContent = el.dataset[currentLang] || el.dataset.zh;
  });
  var btn = document.getElementById('langToggleBtn');
  if (btn) btn.textContent = currentLang === 'zh' ? 'မြ' : '中';
  if (typeof allItems !== 'undefined' && allItems.length) renderItems();
}
```

### 初始化

页面 `window.addEventListener('load', ...)` 里调用 `applyLang()`，确保首次加载就应用正确语言。

## 翻译覆盖范围

| 区域 | 处理方式 | 覆盖 |
|------|---------|------|
| 顶栏（标题、按钮） | `data-zh/my` 属性 | ✅ |
| 底栏（提交按钮） | `data-zh/my` 属性 | ✅ |
| 进度文字 | `t(key)` in JS | ✅ |
| 加载状态 / 空状态 | `t(key)` in JS | ✅ |
| 确认弹窗、退出弹窗 | `data-zh/my` 属性 | ✅ |
| 新增物品表单（标签、占位符、按钮） | `data-zh/my` 属性 | ✅ |
| 单位下拉选项（15 项） | `data-zh/my` 属性 | ✅ |
| 类别下拉选项（18 项） | `data-zh/my` 属性 | ✅ |
| Toast 提示 | `t(key)` in JS | ✅ |
| 物品卡片文字（"现有:" 等） | `makeCard()` 内 `t(key)` | ✅ |
| **分类 Tab 名称** | 来自 Google Sheet，不翻译 | ⚠️ 保持原样 |

> 分类 Tab（肉类、菜类…）由服务端动态返回，前端无法翻译。不影响日常使用。

## UI 设计

### 切换按钮

顶栏右侧，退出按钮左侧：

```html
<button id="langToggleBtn" class="lang-toggle-btn" onclick="setLang(currentLang==='zh'?'my':'zh')">မြ</button>
```

**CSS：**
```css
.lang-toggle-btn {
  background: rgba(255,255,255,0.2);
  color: #fff;
  border: 1px solid rgba(255,255,255,0.4);
  border-radius: 6px;
  padding: 4px 9px;
  font-size: 13px;
  cursor: pointer;
  font-family: inherit;
  line-height: 1.4;
}
```

**显示逻辑：**
- 当前中文 → 按钮显示 `မြ`（点击切换到缅甸语）
- 当前缅甸语 → 按钮显示 `中`（点击切换回中文）

### 视觉效果

```
中文模式：  [ 🔥 库存盘点 ]  Ah kaung  [မြ]  [退出]
缅甸语模式：[ 🔥 ကုန်ပစ္စည်းစစ်ဆေး ]  Ah kaung  [中]  [ထွက်]
```

## 文件改动清单

**`CheckPage.html` 唯一修改文件：**

1. **CSS** — 新增 `.lang-toggle-btn` 样式
2. **HTML** — 顶栏加 `<button id="langToggleBtn">` + 静态元素加 `data-zh/my` 属性（约 40 处）
3. **JS** — 新增 `STRINGS` 字典 + `t()` + `setLang()` + `applyLang()` + 初始化调用
4. **JS makeCard()** — "现有:" 等字符串换成 `t(key)`
5. **JS 动态字符串** — toast、showStatus、弹窗文字换成 `t(key)`（约 30 处）
